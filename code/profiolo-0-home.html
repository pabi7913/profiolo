Service

  • DESIGN

    美術設計

    以客戶需求為出發點，尋找合適的素材製作各種風格的設計。期許自己能達到不用解釋的設計，並讓人留下印象，會心的一笑，並要求自己比上次更好。
    熟悉軟體為Photoshop、Illustrator、InDesign、XD，對排版有執著，對齊一致性是基本。對色彩有感情，喜歡colorful的事物，希望為這世界多添一點色彩。

  • CODING

    程式編碼

    因應無紙化來臨，希望透過網頁互動，將作品更快速且精準的傳達，並作出相應的互動體驗介面，與使用者建立更深層的交互關係。
    運用技術及程式有Html5, CSS3, VS Code, Jquery, Bootstrap。不喜歡看書，卻很喜歡看工具書，平時喜歡自學新的知識及技巧，遇到問題總是想辦法要解決不然會睡不著覺。

  • DIRECTION

    溝通協調

    「耐心」是因為想讓案件順利執行，會仔細聆聽所有需求，並盡力達成目標。「細心」是因為不想重複做一樣的是，會一次做到好，並標示清楚，以便備查。「同理心」是因為從事過服務業，會希望所有客戶都能得到好的對待。
    並以早餐店阿姨為榜樣，會記錄所有客戶的喜好及地雷，希望讓所有人都滿意。
